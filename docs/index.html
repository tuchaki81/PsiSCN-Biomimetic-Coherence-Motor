About

This site is generated automatically from the simulation pipeline. On each push to the default branch, GitHub Actions can run the coherence-criticality code, collect the output figures, publish them to GitHub Pages, and expose the latest summary JSON alongside the plots.

Simulation

Runs your 2D TDGL / coherence criticality code from run_simulation.py.

Publication

Copies generated figures into the Pages bundle and writes a lightweight manifest for the site.

Deployment

Deploys automatically using GitHub Actions and the official Pages actions.

Automation Flow

  1. Push code to the repository.
  2. GitHub Actions installs dependencies from requirements.txt.
  3. The workflow runs python run_simulation.py.
  4. The helper script copies the generated figures into docs/outputs/.
  5. The site is deployed automatically to GitHub Pages.
